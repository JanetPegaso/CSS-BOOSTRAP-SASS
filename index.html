Las pseudoclases es una serie de selectores que aplican a las etiquetas, clases, id en css para que actúen de una manera determinada (para que se le añadan o modifiquen sus estilos)

Pseudo clase HOVER

Es una clase que se activa cuando tenemos el ratón encima de la etiqueta X
